・VSCode プラグイン
go

Ctrl-Shift-P Install/Update Tools 全部チェックを入れてインストール
Ctrl-Shift-P Get Package importで記載したモジュールをインストールする

・バイナリ作成
go install (src配下のディレクトリ名)

・テスト実行
※ソースとテストファイルのディレクトリに移動すること

（テスト実行）
go test

(テスト実行_結果詳細)
go test -v

（テスト実行_カバレッジ出力）
go test -cover

（特定のテストのブロックを実行する）
go test -short
以下のブロック内を実行する
if testing.Short() {}

（テスト実行_並列実行）
go test --paralell 3
testing.T.parallel呼び出しのあるテストを並列実行する。
